Arduino Part:
- Contains 2 sensors, a light dependant resistor and temperature sensor (lm35). Contains 2 LEDs with different colours.

- System will detect the surrounding light intensity and temperature and the corresponding LED will light up.

Pin References:
Digital 13 - Temperature LED (Yellow)
Digital 12- LDR LED (Red)

Analog 0 - LDR Sensor
Analog 1 - Temperature Sensor (lm35)
